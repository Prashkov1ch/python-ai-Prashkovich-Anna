# Your data / Ваши данные

## RU

В эту папку вы помещаете СВОИ данные в формате CSV, которые будете анализировать в ноутбуках.

Требования:
- Формат: CSV, кодировка UTF-8, разделитель запятая.
- Минимум 50 строк и 3 столбца.
- Желательно реальные данные (например, из Wikidata или других открытых источников).
- Перед началом работы согласуйте тему и источник данных с преподавателем.
