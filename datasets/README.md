# Datasets (examples from the teacher)

## RU: Требования к CSV-файлам

- Формат файла: CSV (comma-separated values).
- Кодировка: UTF-8 (важно для корректного отображения русского текста и специальных символов).
- Разделитель: запятая ','.
- Первая строка: заголовки столбцов (без лишних пробелов).
- Минимум 10–20 строк для учебных примеров.
- Данные должны выглядеть реалистично, а не как случайный мусор.

## EN: CSV requirements

- File format: CSV (comma-separated values).
- Encoding: UTF-8 (to support Cyrillic and international characters correctly).
- Delimiter: comma ','.
- First row: column headers.
- At least 10–20 rows for example datasets.
- Data should look realistic, not random gibberish.

Эти файлы можно использовать как пример структуры данных для своих проектов.
/ You can use these files as structural examples for your own datasets.
